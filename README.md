什么是Node.js?
	官网描述
	Node.js是一个基于Chrome V8引擎的JavaScript运行环境
	Node.js使用了一个事件驱动 非阻塞式I/O的模型 轻量高效

在Node.js里运行JS和在Chrome里运行JS有什么不同
	几乎没有不一样
	Node.js没有浏览器API document window
	Node.js让你用类似的方式，控制整个计算机

Node.js能用来做什么？
	Web服务
		搜索引擎优化 + 首屏速度优化 = 服务端优化
		服务端渲染 + 前后端同构 = Node.js

	可扩展性比较好



BFF层(Backend for Frontend)
	对用户侧提供HTTP服务
	使用后端RPC服务
